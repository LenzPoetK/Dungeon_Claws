# Dungeon_Claws


  === Objetivo ===
O objetivo do projeto foi o fortalecimento dos conhecimentos adquiridos durante o ensino da linguagem.
Isto foi alcançado por meio de uma abordagagem de sprints, na qual toda semana uma parte significativa 
do jogo é feita e testada. O conceito,apesar de simples, possibilitou um avanço significativo na
compreensão de alguns conceitos chave. Em especial: Práticas de código organizado e compreensível para 
os demais componentes, matrizes, funções e ponteiros.

  === Beyond Project ===
o jogo final possui 3 fases com complexidade e dificuldade crescentes, possuindo diversos obstáculos 
para desafiar o jogador. Como espinhos mortais, inimigos que vagam perdidos pelo mapa ou te perseguem.
Além de quebra-cabeças baseados na dedução e exploração do mapa por parte do jogador.


o jogo se passa em 1989, num contexto suburbano, onde o o governo faz experimentos secretos afim de se
comunicar com o além. porém um dia algo dá errado e cabe ao protagonista descobrir o que aconteceu. 

Este jogo é um projeto feito por josé Victor colino, leonardo gomes e andré buna para a matéria de programação.
