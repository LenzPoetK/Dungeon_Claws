# Dungeon_Claws


  === Objetivo ===
O objetivo do projeto foi o fortalecimento dos conhecimentos adquiridos durante o ensino da linguagem.
Isto foi alcançado por meio de uma abordagagem de sprints, na qual toda semana uma parte significativa 
do jogo é feita e testada. O trabalho 


o jogo possui 3 fases com complexidade crescente, possuindo diversos obstáculos para desafiar o jogador.


o jogo se passa em 1989, num contexto suburbano, onde o o governo faz experimentos secretos afim de se comunicar com o além. porém um dia algo dá errado e cabe ao protagonista descobrir o que aconteceu. 

Este jogo é um projeto feito por josé Victor colino, leonardo gomes e andré buna para a matéria de programação.
